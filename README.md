## Quick setup
- Install Node JS
- Create an .env file: 
    - Copy contents of `.env.example` to a new `.env` file
- Run the following commands in the extracurricular folder to install dependencies:
```sh
npm install
```
- Run the following command to start the app
```sh
npm run dev
```
